# RuneScape Blog (HTML, CSS e PHP)

Projeto de blog desenvolvido em HTML, CSS e PHP puro para praticar:
- Integração PHP + HTML
- Organização de layout com includes
- Navegação entre páginas
- Filtro de posts por categoria
- Ordenação por data
- Estruturação de CSS

## Tecnologias
- PHP
- HTML5
- CSS3

## Objetivo
Projeto criado com fins de estudo e prática de desenvolvimento web backend com PHP.
